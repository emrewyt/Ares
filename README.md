```
1. Ares 2.0.0 arşivini indirin

```

```
5. "İndir" butonuna tıklayın
6. Steam'den oyunu indirin
7. "Kur" butonuna tıklayın
8. Steam'i yeniden başlatın
9. Oyununuz kütüphanenizde hazır!
```
